Função simples para verificar se um produto está vencido ou não.
a função recebe como parametro a data de fabricação e uma string informando o tempo  de validade em dias. É feita a comparação entre a fabricação + o tempo decorrido e a data atual.
Se a fabricação + tempo decorrido for maior que a data atual o produto esta dentro do prazo de validade, se não, está vencido.
